# Jimmy's Diner - Restaurant Ordering App

A web-based ordering app for a restaurant, built with vanilla HTML, CSS, and JavaScript.

## Features
- Dynamic menu rendering from a JavaScript data source
- Add and remove items from the cart
- Real-time total price calculation
- Checkout modal with form validation

## Technologies
- HTML5
- CSS3 (Flexbox)
- Vanilla JavaScript
- Figma (design spec)
